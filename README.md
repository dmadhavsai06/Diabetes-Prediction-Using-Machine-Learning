# Diabetes-Prediction-Using-Machine-Learning
A Machine Learning project that predicts whether a person has diabetes using the PIMA Indians Diabetes Dataset. Built with Python, Pandas, NumPy, and Scikit-learn using a Support Vector Machine (SVM) model.

## Project Overview
This project predicts whether a person is diabetic or not using Machine Learning.

## Dataset
- PIMA Indians Diabetes Dataset
- Features:
  - Pregnancies
  - Glucose
  - BloodPressure
  - SkinThickness
  - Insulin
  - BMI
  - DiabetesPedigreeFunction
  - Age

## Algorithm Used
- Support Vector Machine (SVM)

## Libraries
- Python
- NumPy
- Pandas
- Scikit-learn

## Accuracy
- Training Accuracy: 0.786%
- Testing Accuracy: 0.772%

## How to Run
1. Clone the repository
2. Install the requirements
3. Open `diabetes_prediction.ipynb`
4. Run all cells
